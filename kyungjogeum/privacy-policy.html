경조금 기록기 개인정보 처리방침

시행일: 2026년 6월 15일
최종 수정일: 2026년 6월 15일

경조금 기록기(이하 “앱”)는 사용자의 경조금 기록과 경조사 일정을 사용자가 직접 관리할 수 있도록 제공되는 Android 앱입니다. 본 개인정보 처리방침은 앱이 어떤 정보를 어떤 목적으로 처리하는지, 정보가 어디에 저장되는지, 사용자가 어떤 선택권을 가지는지 설명합니다.

앱의 주요 기록 데이터는 기본적으로 사용자의 기기 내부에 저장됩니다. 앱은 자체 회원가입, 자체 서버 저장, 자체 계정 동기화 기능을 제공하지 않습니다.

1. 처리하는 개인정보 항목

앱은 사용자가 직접 입력하거나 선택한 경우 다음 정보를 기기 내부에 저장할 수 있습니다.

  • 사람 정보: 이름, 연락처, 연락처 없음 여부
  • 경조금 기록: 보냄/받음 구분, 경조사 종류, 세부 선택, 경조사명, 금액, 참석 여부, 날짜, 메모
  • 경조사 일정: 일정명, 날짜, 사람 이름, 연락처, 경조사 종류, 장소, 메모, 알림 설정
  • CSV 가져오기/내보내기 결과 및 사용자가 선택한 CSV 파일의 내용

앱은 사용자가 연락처에서 사람을 불러오도록 선택한 경우, 선택한 연락처의 이름과 전화번호를 기록 입력 편의를 위해 사용할 수 있습니다.

2. 개인정보 처리 목적

앱은 다음 목적을 위해 개인정보를 처리합니다.

  • 사람별 경조금 지급 및 수령 기록 저장
  • 동명이인 구분과 사람별 기록 통합
  • 경조금 기록 검색, 합계 계산, 대시보드 통계 표시
  • 경조사 일정 관리 및 사용자가 설정한 전날/당일 알림 제공
  • CSV 내보내기와 가져오기를 통한 사용자의 데이터 백업 및 복원
  • 하단 배너 광고 표시와 광고 품질, 보안, 부정행위 방지

3. 권한 사용

앱은 다음 Android 권한을 사용할 수 있습니다.

  • 연락처 권한: 사용자가 “연락처에서 불러오기”를 선택했을 때 기기 연락처에서 이름과 전화번호를 선택하기 위해 사용합니다. 권한을 허용하지 않아도 이름과 연락처를 직접 입력하거나 연락처 없음으로 기록할 수 있습니다.
  • 알림 권한: 사용자가 경조사 일정의 전날 또는 당일 알림을 설정했을 때 알림을 표시하기 위해 사용합니다. 권한을 허용하지 않아도 경조금 기록과 일정 저장 기능은 사용할 수 있습니다.

4. 저장 위치와 외부 전송

사용자가 입력한 경조금 기록, 사람 정보, 경조사 일정은 기본적으로 사용자의 기기 내부 앱 저장소에 저장됩니다. 앱은 자체 서버로 이 기록을 업로드하거나 자체 서버에서 동기화하지 않습니다.

CSV 내보내기 기능을 사용하면 사용자가 선택한 위치에 CSV 파일이 생성됩니다. CSV 파일에는 사용자가 앱에 저장한 이름, 연락처, 금액, 날짜, 메모 등이 포함될 수 있으므로 사용자가 직접 안전하게 관리해야 합니다.

사용자의 Android 시스템 백업 설정 또는 기기 제조사/운영체제 기능에 따라 앱 데이터가 기기 백업 대상에 포함될 수 있습니다. 이러한 백업은 앱이 직접 운영하는 서버 저장 기능이 아니며, 사용자의 기기 및 계정 설정에 따릅니다.

5. 광고 및 제3자 SDK

앱은 무료 서비스 제공을 위해 Google AdMob 하단 배너 광고를 표시할 수 있으며, Google Mobile Ads SDK를 사용합니다.

Google Mobile Ads SDK는 광고 제공, 분석, 앱 성능 진단, 보안 및 부정행위 방지 목적을 위해 IP 주소, 앱 상호작용 정보, 진단 정보, 기기 또는 계정 식별자(예: 광고 ID, 앱 세트 ID 등)를 자동으로 수집하거나 Google에 전송할 수 있습니다. 해당 데이터는 Google의 정책에 따라 처리됩니다.

광고 식별자는 사용자의 Android 설정에서 재설정하거나 삭제할 수 있습니다. 광고 및 Google의 데이터 처리에 관한 자세한 내용은 Google 정책과 AdMob 안내를 참고하시기 바랍니다.

6. 개인정보 보관 및 삭제

앱에 저장된 기록은 사용자가 앱 안에서 삭제하거나, 기기 설정에서 앱 데이터를 삭제하거나, 앱을 삭제할 때 삭제될 수 있습니다.

  • 경조금 기록과 경조사 일정: 사용자가 앱에서 수정 또는 삭제할 때까지 기기 내부에 보관됩니다.
  • CSV로 내보낸 파일: 앱 내부 데이터와 별개로 사용자가 저장한 위치에 남을 수 있으며, 삭제는 사용자가 직접 수행해야 합니다.
  • 광고 SDK 관련 데이터: Google Mobile Ads SDK 및 Google 정책에 따라 처리됩니다.

앱은 자체 회원가입 계정을 제공하지 않으므로 별도의 계정 삭제 절차는 없습니다.

7. 개인정보의 제3자 제공

앱은 사용자가 입력한 경조금 기록, 사람 정보, 경조사 일정 정보를 개발자가 운영하는 자체 서버에 제공하거나 판매하지 않습니다.

다만, 광고 표시를 위해 Google Mobile Ads SDK가 광고 관련 데이터를 Google에 전송할 수 있으며, 사용자가 직접 CSV 파일을 외부 앱이나 저장소로 공유하는 경우 해당 공유는 사용자의 선택에 따라 이루어집니다.

8. 아동의 개인정보

앱은 아동을 주요 대상으로 제공되지 않으며, 아동으로부터 고의로 개인정보를 수집하기 위한 기능을 제공하지 않습니다.

9. 보안

앱은 사용자가 입력한 데이터를 Android 앱 저장소에 보관합니다. 제3자 SDK가 네트워크로 데이터를 전송하는 경우에는 해당 SDK의 보안 전송 정책에 따릅니다. 사용자는 기기 잠금, 운영체제 업데이트, CSV 파일 관리 등을 통해 자신의 데이터를 보호해야 합니다.

10. 개인정보 처리방침 변경

앱 기능, 광고 SDK, 법령 또는 정책 변경에 따라 본 개인정보 처리방침이 수정될 수 있습니다. 중요한 변경이 있는 경우 본 페이지의 최종 수정일을 갱신합니다.

11. 문의

개인정보 처리와 관련한 문의는 Google Play 스토어에 표시된 개발자 연락처를 통해 접수할 수 있습니다.
