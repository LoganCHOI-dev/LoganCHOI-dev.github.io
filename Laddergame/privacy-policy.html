사다리타기 개인정보 처리방침

사다리타기: 복불복, 뽑기, 점심내기, 벌칙(이하 “앱”)은 사용자가 직접 참가자와 결과를 입력하고 사다리타기 결과를 확인할 수 있도록 제공되는 Android 앱입니다. 본 개인정보 처리방침은 앱이 어떤 정보를 어떤 목적으로 처리하는지, 정보가 어디에 저장되는지, 사용자가 어떤 선택권을 가지는지 설명합니다.

앱의 주요 입력 데이터는 기본적으로 사용자의 기기 내부에 저장됩니다. 앱은 자체 회원가입, 자체 로그인, 자체 서버 저장, 자체 계정 동기화 기능을 제공하지 않습니다.

1. 처리하는 개인정보 항목

앱은 사용자가 직접 입력하거나 선택한 경우 다음 정보를 기기 내부에 저장할 수 있습니다.

  • 참가자 이름
  • 결과 또는 벌칙 문구
  • 실행 속도, 공개 방식, 가림막 사용 여부 등 게임 설정
  • 결과 이미지 공유를 위해 사용자가 선택한 결과 목록

앱은 연락처, 위치, 카메라, 사진, 마이크, 캘린더, 건강 정보, 금융 정보 접근 권한을 요구하지 않습니다. 사용자가 참가자 이름이나 결과 문구에 실제 이름, 연락처, 주소, 민감한 내용을 직접 입력하는 경우 해당 내용은 기기 내부 앱 저장소에 저장될 수 있습니다.

2. 개인정보 처리 목적

앱은 다음 목적을 위해 정보를 처리합니다.

  • 참가자와 결과/벌칙 목록 표시
  • 사다리 생성, 경로 계산, 결과 매칭
  • 사용자가 설정한 게임 상태 복원
  • 결과 이미지 생성 및 사용자가 선택한 외부 앱으로 공유
  • 앱 공유하기, 앱 평가하기, 개인정보 처리방침 열기 등 사용자가 선택한 외부 이동 처리
  • 하단 배너 광고, 전면 광고 표시와 광고 품질, 보안, 부정행위 방지

3. 권한 사용

앱 자체 기능은 연락처, 위치, 카메라, 사진, 마이크 같은 민감 권한을 요청하지 않습니다.

광고 표시를 위해 Google Mobile Ads SDK가 네트워크 연결, 광고 ID, Android 광고 서비스 관련 권한을 사용할 수 있습니다. 이는 Play Console 앱 콘텐츠의 광고 및 데이터 보안 선언과 일치합니다.

4. 저장 위치와 외부 전송

사용자가 입력한 참가자 이름, 결과/벌칙 문구, 게임 설정은 기본적으로 사용자의 기기 내부 앱 저장소에 저장됩니다. 앱은 자체 서버로 이 정보를 업로드하거나 자체 서버에서 동기화하지 않습니다.

결과 이미지 공유 기능을 사용하면 앱은 공유용 PNG 이미지를 기기 캐시 영역에 생성하고 Android 공유 시트를 엽니다. 사용자가 외부 앱을 선택해 공유한 이후의 처리는 해당 외부 앱 또는 서비스의 개인정보 처리방침을 따릅니다.

사용자의 Android 시스템 백업 설정 또는 기기 제조사/운영체제 기능에 따라 앱 데이터가 기기 백업 대상에 포함될 수 있습니다. 이러한 백업은 앱이 직접 운영하는 서버 저장 기능이 아니며, 사용자의 기기 및 계정 설정에 따릅니다.

5. 광고 및 제3자 SDK

앱은 무료 서비스 제공을 위해 Google AdMob 광고를 표시할 수 있으며, Google Mobile Ads SDK를 사용합니다.

Google Mobile Ads SDK는 광고 제공, 광고 성과 측정, 분석, 앱 성능 진단, 보안 및 부정행위 방지 목적을 위해 다음 데이터를 자동으로 수집하거나 Google 및 광고 파트너와 공유할 수 있습니다.

  • 대략적인 위치: IP 주소 등을 기반으로 추정되는 일반적인 위치 정보
  • 앱 상호작용: 앱 실행, 탭, 광고 노출, 광고와의 상호작용 정보
  • 진단 정보: 앱 또는 SDK 성능, 오류, 지연, 에너지 사용량 등 진단 정보
  • 기기 또는 기타 ID: Android 광고 ID, 앱 세트 ID 등 광고와 부정행위 방지에 사용될 수 있는 식별자

광고 식별자는 사용자의 Android 설정에서 재설정하거나 삭제할 수 있습니다. 광고 및 Google의 데이터 처리에 관한 자세한 내용은 Google Mobile Ads SDK 데이터 공개 안내와 Google 개인정보처리방침을 참고하시기 바랍니다.

6. 개인정보 보관 및 삭제

앱에 저장된 참가자 이름, 결과/벌칙 문구, 게임 설정은 사용자가 앱 안에서 수정하거나, 기기 설정에서 앱 데이터를 삭제하거나, 앱을 삭제할 때 삭제될 수 있습니다.

  • 게임 입력 데이터: 사용자가 수정 또는 삭제하거나 앱 데이터가 삭제될 때까지 기기 내부에 보관됩니다.
  • 공유용 결과 이미지: 앱 캐시 영역에 저장되며, Android 시스템 또는 사용자의 저장공간 정리 동작에 따라 삭제될 수 있습니다.
  • 광고 SDK 관련 데이터: Google Mobile Ads SDK 및 Google 정책에 따라 처리됩니다.

앱은 자체 회원가입 계정을 제공하지 않으므로 별도의 계정 삭제 절차는 없습니다.

7. 개인정보의 제3자 제공

앱은 사용자가 입력한 참가자 이름, 결과/벌칙 문구, 게임 설정을 개발자가 운영하는 자체 서버에 제공하거나 판매하지 않습니다.

다만, 광고 표시를 위해 Google Mobile Ads SDK가 광고 관련 데이터를 Google에 전송할 수 있으며, 사용자가 직접 결과 이미지나 앱 링크를 외부 앱으로 공유하는 경우 해당 공유는 사용자의 선택에 따라 이루어집니다.

8. 아동의 개인정보

앱은 아동을 주요 대상으로 제공되지 않으며, 아동으로부터 고의로 개인정보를 수집하기 위한 회원가입, 로그인, 서버 저장 기능을 제공하지 않습니다.

9. 보안

앱은 사용자가 입력한 데이터를 Android 앱 저장소에 보관합니다. 제3자 SDK가 네트워크로 데이터를 전송하는 경우에는 해당 SDK의 보안 전송 정책에 따릅니다. 사용자는 기기 잠금, 운영체제 업데이트, 공유 이미지 관리 등을 통해 자신의 데이터를 보호해야 합니다.

10. 개인정보 처리방침 변경

앱 기능, 광고 SDK, 법령 또는 정책 변경에 따라 본 개인정보 처리방침이 수정될 수 있습니다. 중요한 변경이 있는 경우 본 페이지를 통해 변경된 내용을 안내합니다.
